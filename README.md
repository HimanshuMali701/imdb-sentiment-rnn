# 🎬 IMDb Sentiment Analysis (RNN)

AI-powered sentiment analysis web app that classifies IMDb movie reviews as **positive** or **negative** using a Recurrent Neural Network (RNN) and Streamlit.

---

## 🚀 Features

* Predict sentiment of movie reviews (Positive / Negative)
* RNN-based deep learning model
* Text preprocessing & tokenization
* Interactive Streamlit web app

---

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* NLP (Tokenization, Padding)
* Streamlit
* Hugging Face Spaces
* Docker

---

## ▶️ Run Locally

```bash
git clone https://github.com/HimanshuMali701/imdb-sentiment-rnn.git
cd imdb-sentiment-rnn
pip install -r requirements.txt
streamlit run app.py
```
---
## 📦 Deployment

This project is deployed on Hugging Face Spaces using a Docker-based Streamlit setup, solving dependency limitations (like TensorFlow support in standard environments).
